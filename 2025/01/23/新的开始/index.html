博客

缘起

想做一个自己的博客是学期中就产生的想法了，但是由于拖延症一直没能开始做这件事情。考完期末以后想着手开始搞，但是一直拖拖拉拉到现在才基本完成。做博客的目的主要是希望能有个地方记录自己学习的过程，以及放置一些无处放置的想法，或者让偶尔突然产生的创作欲能够落脚。将博文放到网上而不是留存在本地，或许能让我更有动力做这件事情。另外其实也有一些从众的意思在里面。
出于以上目的，我可能并不会主动宣传这个博客，或者在什么地方贴上这个博客的链接，而更希望把它当作一片无人来访的自留地。所以，博客里后续的文章可能会比较随意，写到哪里算哪里，不会有一定的规范。如果有幸运的读者看到这里，欢迎在博客里随便逛逛！不用在意内容，看个乐子就行。

建立博客

一开始高估了建立博客的难度，所以迟迟没有下手。后来了解了一下，似乎不用很复杂，而且我对博客的要求也不高，不需要花里胡哨的功能，能够发布文章就行。于是1月20号凌晨就开始上手。
首先研究了一下不同路线，选择了Hexo的框架，并用Github Pages托管。然后对着网上的教程安装好node.js和hexo，并完成初始化，这时立即就可以通过localhost在浏览器上看到自己的博客了。这其实就是几行命令的事情，花了一个小时都不到，确实比我想象的快很多。
接下去就要把博客托管到Github Pages上去。只要在github里建一个仓库，在_config.yml文件里指明参数，用一行命令就可以完成托管。这里遇到了一个bug，hexo d一直提示找不到仓库。一开始还以为代理有问题，折腾了半天才发现其实是_config.yml里面有个.com写成了.io。改过来之后一下就对了。此时，已经可以通过url访问博客了。因为目前的url看着还算顺眼，暂时还没有花钱买域名的想法。

装修博客

现在的博客虽然能用，但实在有点丑。虽然对博客的要求仅限于能用就好，但我有点强迫症，还是想花时间稍微弄一弄外观。没想到一弄就是三四天。
首先我想把theme从默认的landscape换掉。在一众主题里稍微挑了挑，最后选中了Cactus。主要原因是偏好简洁的风格，不想弄一个太热门的主题（比如next），另一层因素是单纯比较喜欢这个主题的名字。
装好主题后又对默认的logo、字体等等不满意，于是找了个logo网站随便弄了几个logo出来，再在themes/cactus/里面的各种配置文件里寻找参数来进一步优化。由于对前端的知识不熟悉，也不会写HTML之类，所以这个过程比较漫长。总结一下大概是这样：

  • 修改网站内容一般在.ejs里面找。因为不太会调用定义好的变量，想要添加内容的时候干脆直接用常量代替，这样虽然代码丑了点，但博客看起来没有区别。如index.ejs里面就可以修改博客主页的内容。
  • 修改字体、颜色等格式一般在.styl里面找。修改主题颜色在white.styl中，注册字体在fonts.styl中，修改博文界面的参数在article.styl中等。除了修改数值，也可以自己添加新的代码行。试一试一般都可以正常工作，比较简单。包括markdown在网页端的显示效果也可以通过修改.styl文件实现。正文字体最终选择了思源黑体，主要因为它是一种类似微软雅黑的无衬线字体，而且英文字母看起来比微软雅黑顺眼一些。
  • 最后是cactus文件夹下面的_config.yml文件。它的功能实际上基本被上面两种文件包含了，只是有时候用它会更省事一些。我用它开启了博文的tag功能，并打开MathJax以支持数学公式（不知道为什么这个默认是false）。顺便一提，还实现了搜索功能，其实也是一行命令的事。
  • 有很多可实现的功能并没有实现，这背后有不少考虑（主要还是懒）。如评论系统、访问统计、主页更多的栏目（如Projects）、友链等等。也没有指向个人主页的链接，因为还没有个人主页。希望以后有机会能有一个正经的个人主页。这些以后可能会去完善，甚至重构整个博客，但至于现在，能用就行。
    总之，如果有什么想修改的地方直接到源码里去找就行，感觉还是比较好上手的，没有太大的障碍，也不会出现很严重的bug什么的。改完以后，hexo clean && hexo g && hexo d && hexo s就可以在本地实时预览了。记得不要忘了hexo clean，否则新旧文件混在一起可能导致显示出问题。
    最后是博客的取名。因为没什么人会看到博客，所以也没有在名字上花太多的心思。取名Phase Space主要是以下考虑：一是直译为物理中的相空间，二是phase暗指月相，和博客的logo相合，三是space既有太空之意对应月亮，也是个人空间的指代。其实解释了这么多，起这个名字只是拍脑袋的想法，没有太多的研究在里面。毕竟是自留地，能看就行。

相空间里有什么?

我的创作欲和表达欲可能并不是非常旺盛，因此不会将大小琐事都发到博客上。就目前而言，可能博客里只会发一些读书、读paper或者课程的笔记之类比较严肃和无聊的东西，而不会有记录生活的文章。当然，也许以后某天我会突然想写点不一样的而改变这个计划。谁知道呢？
